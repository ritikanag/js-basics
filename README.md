# JavaScript Basics

This repository contains JavaScript programs that focus on understanding and practicing **core JavaScript fundamentals**.  
The goal of this repository is to strengthen problem-solving skills and build a solid foundation in JavaScript by implementing programs related to different language concepts.

---

## Topics Covered

The programs in this repository are organized based on JavaScript concepts such as:

- Variables & Data Types
- Operators
- Conditional Statements
  - if
  - if-else
  - switch
- Loops
  - for loop
  - while loop
  - do...while loop
- Functions
- Arrays
- Strings
- Objects
- Basic Algorithms
- Pattern Programs
- Logical Interview Questions


## Purpose of this Repository

- Practice **JavaScript fundamentals**
- Improve **logic building skills**
- Prepare for **technical interviews**
- Maintain a **structured collection of JS programs**

---

## Tech Used

- JavaScript (ES6+)
- Node.js (for running programs locally)
